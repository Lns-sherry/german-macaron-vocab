## 🌸 About the Project | Über das Projekt | 关于项目

**English:** Welcome to **German Macaron Vocab**! Forget those dry, gray flashcards. We’ve turned the "bitter" struggle of German vocabulary into a "sweet" daily treat. This app, born in **Google AI Studio** and powered by **Gemini AI**, makes learning feel like a visit to a pastel-colored patisserie.

**Deutsch:** Willkommen bei **German Macaron Vocab**! Vergiss trockene, graue Karteikarten. Wir haben das „bittere“ Vokabelpauken in ein „süßes“ tägliches Vergnügen verwandelt. Diese App, entstanden im **Google AI Studio** und unterstützt durch **Gemini KI**, lässt das Lernen wie einen Besuch in einer pastellfarbenen Konditorei erscheinen.

**中文：** 欢迎来到 **German Macaron Vocab**！告别枯燥沉闷的灰度单词卡片。我们将“痛苦”的德语背诵变成了一场甜蜜的每日治愈。本项目诞生于 **Google AI Studio**，由 **Gemini AI** 驱动，让学习德语的过程就像逛一间清新可爱的马卡龙甜点店。

## 🚀 How to Enjoy | Zum Mitmachen | 如何开始使用

1.  **Clone the recipe:**
    ```bash
    git clone [https://github.com/Lns-sherry/german-macaron-vocab.git](https://github.com/Lns-sherry/german-macaron-vocab.git)
    ```
2.  **Prepare the ingredients:**
    ```bash
    npm install
    ```
3.  **Add the secret sauce (API Key):**
    * Rename `.env.example` to `.env`.
    * Add your own [Google AI Studio API Key](https://aistudio.google.com/).
4.  **Start baking:**
    ```bash
    npm run dev
    ```
